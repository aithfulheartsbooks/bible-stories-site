# ChatGPT Website Instructions

Purpose

This website is for Faithful Hearts Bible story books.

Goal

Make it EASY to update future books without explaining everything again.

When adding a new book

1. Copy the layout of existing books
2. Use same image size as current covers
3. Keep style consistent
4. Add:
 * Title
 * Description
 * Cover image
 * Amazon link

Cover images

* Do NOT stretch images
* Match size of Book 1 or Book 2 cover
* Keep square ratio if used

Writing style

* Simple
* Warm
* Christian / family-friendly
* Easy for kids and parents

Before finishing changes

* Check images load
* Check links work
* Check mobile view
* No grammar mistakes

IMPORTANT

Always follow existing website structure.
Do not redesign unless asked.
